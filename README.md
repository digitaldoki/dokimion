## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repo (e.g., `digitaldokimion.com`)
2. Push this folder contents to the `main` branch
3. Go to **Settings → Pages → Source** → select `main` / `/ (root)` → Save
4. Add custom domain `digitaldokimion.com` in Settings → Pages

### DNS Records for Custom Domain
```
A    @    185.199.108.153
A    @    185.199.109.153
A    @    185.199.110.153
A    @    185.199.111.153
CNAME  www  yourusername.github.io
```
Enable **Enforce HTTPS** once DNS propagates (24-48 hrs).
